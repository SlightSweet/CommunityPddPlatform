# CommunityPddPlatform

社区团购平台项目

## 项目简介
CommunityPddPlatform 是一个基于 Spring Boot 的社区团购系统，包含团长管理、商品管理、订单管理、售后服务等模块。

## 模块说明
- `leader-service`：团长入驻与管理
- `group-order-service`：团购订单处理
- `group-product-service`：商品管理与秒杀
- `group-service`：团购核心服务
- `after-sales-service`：售后处理
- `common`：公共工具与配置

## 技术栈
- Java 1.8
- Spring Boot 2.3.12
- MyBatis 2.2.0
- MySQL 8.0.33
- Lombok
